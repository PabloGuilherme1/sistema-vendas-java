# Sistema de Vendas em Java (Console)

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos.
O sistema simula um fluxo básico de vendas para pequenos varejos, utilizando
execução via terminal.

## Funcionalidades
- Cadastro de produtos
- Registro de vendas
- Cálculo automático do total da venda
- Exibição dos itens vendidos no terminal

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
- Git e GitHub

## Como executar
1. Clone o repositório
2. Importe o projeto em uma IDE Java (Eclipse, IntelliJ)
3. Execute a classe `Program`
